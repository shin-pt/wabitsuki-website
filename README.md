## 🚧 未実装・今後の対応事項

### 要対応（高優先度）

1. **LINE公式アカウントのURL設定**
   - 現在 `https://lin.ee/placeholder` がプレースホルダーとして設定されています
   - LINE公式アカウント（@237kyhbi）の正しい友だち追加URLに置換が必要です
   - 全ファイルで `https://lin.ee/placeholder` を一括置換してください

2. **Googleマップ埋め込みの正確な設定**
   - 現在は座標ベースの仮埋め込みです
   - Googleマイビジネスに登録後、正式な埋め込みコードに差し替えを推奨

3. **実写真の差し替え**
   - 現在はプレースホルダー（アイコン表示）です
   - 以下の写真が必要：
     - 施術者（田中孝明）の写真
     - 院内・内装の写真
     - 施術風景の写真（プライバシー配慮済み）
     - 各ブログ記事のサムネイル

4. **ロゴデータの実装**
   - ロゴデータあり（はい）とのことですが、現在はテキストロゴです
   - ロゴ画像を提供いただき次第、差し替え可能

### 中優先度

5. **ブログ記事の個別ページ作成**
   - 現在は一覧ページのみ
   - 各記事の詳細ページテンプレートの作成

6. **お客様の声（口コミ）セクション追加**
   - 信頼性向上のため推奨
   - Googleマイビジネスの口コミと連動も検討

7. **初回特典・キャンペーン表示**
   - トップページのCTAに初回特典情報を追加

8. **OGP用画像の設定**
   - SNSシェア時のサムネイル画像

### 低優先度

9. **Google Analytics / Tag Manager の設置**
   - アクセス解析・CV計測

10. **ファビコン（favicon）の設定**
    - ブラウザタブに表示されるアイコン

11. **PWA対応**
    - オフラインアクセス・ホーム画面追加
